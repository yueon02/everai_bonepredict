üìö Understanding the Metrics

Accuracy

Overall percentage of correct predictions out of all predictions made.

Accuracy = (TP + TN) / (TP + TN + FP + FN)

Where: TP=True Positive, TN=True Negative, FP=False Positive, FN=False Negative

Precision

Of all positive predictions, how many were actually correct? Measures prediction accuracy.

Precision = TP / (TP + FP)

High precision = Few false alarms when predicting positive class

Recall (Sensitivity)

Of all actual positive cases, how many did we correctly identify? Measures detection ability.

Recall = TP / (TP + FN)

High recall = Few missed positive cases

F1 Score

Harmonic mean of precision and recall. Balances both metrics into a single score.

F1 = 2 √ó (Precision √ó Recall) / (Precision + Recall)

Best when both precision and recall are high
